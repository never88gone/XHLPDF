# 文件列表
目前文件列表包含三种数据<br>
1.SMB服务器<br>
2.本地文件<br>
3.上传PDF，上传PDF会显示二维码，手机端可以上传PDF的url地址，已经上传手机端本地以及icloud中的PDF文件上传，上传后文件会保存在本地文件中<br>
