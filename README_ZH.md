# TVPDF

<p align="center">
  <img alt="TVPDF" src="screenshot/1920*720.jpg"/>
</p>

TVOS上使用PDF查看工具，可以查看SMB文件系统、本地的PDF文件，同时可以通过手机投屏将url，已经本地文件上传到tv上进行pdf的查看。

# 文件列表

目前文件列表包含三种数据：

1. SMB服务器
2. 本地文件
3. 上传PDF，上传PDF会显示二维码，手机端可以上传PDF的url地址，已经上传手机端本地以及icloud中的PDF文件上传，上传后文件会保存在本地文件中。
